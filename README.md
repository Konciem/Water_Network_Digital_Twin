# Project 10: Simple Digital Twin of Water Infrastructure 

This project focuses on creating a functional prototype of a **Digital Twin** for water infrastructure. It combines network topology (graphs), hydraulic simulations, and a user-friendly dashboard to show how these layers can work together in one tool


## Tech Stack
The following tools are recommended for this project:
* **Simulation:** EPANET or WNTR (Python library).
* **Database:** Neo4j (Graph database for network topology).
* **Language:** Python.
* **Dashboard:** Streamlit or Flask.
